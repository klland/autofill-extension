# 自動填入助手

Chrome 擴充套件，偵測網頁表單欄位並自動填入個人資料。支援多個 Email、各網站帳號密碼、以及常用證件、地址等個人資訊。所有資料僅存於本機，不上傳任何伺服器。

## 功能

- **自動偵測表單欄位** — 點擊輸入框時，自動判斷欄位類型並在上方顯示建議泡泡
- **一鍵填入** — 點「填入」直接填入對應資料
- **多個 Email** — 管理多個信箱，填入時列出所有選項讓你選擇
- **各網站帳號密碼** — 每個網站儲存獨立的帳號密碼，填完後詢問是否記住
- **學習糾正** — 偵測錯誤時可點「其他欄位」手動選擇，下次同網站會記住
- **記憶新資料** — 手動填入新值後，詢問是否更新儲存的資料
- **證件管理** — 支援身分證、台胞證（含到期日）、護照（含到期日）

## 支援欄位

| 類別 | 欄位 |
|---|---|
| 個人資料 | 姓名、生日 |
| 聯絡方式 | 手機號碼、市話號碼 |
| 證件 | 身分證字號、台胞證號碼/到期日、護照號碼/到期日 |
| 地址 | 縣市、鄉鎮區、詳細地址、郵遞區號 |
| 工作 | 公司名稱、職稱 |
| 信箱清單 | 多個電子郵件 |
| 網站帳號 | 各網站帳號密碼 |

## 安裝方式

1. 下載或 clone 此專案
2. 開啟 Chrome，進入 `chrome://extensions/`
3. 開啟右上角「開發人員模式」
4. 點擊「載入未封裝項目」，選擇此專案資料夾
5. 點擊工具列的擴充套件圖示，填入個人資料後點「儲存」

## 資料安全

- 所有資料僅存於本機 `chrome.storage.local`
- 不連線任何伺服器
- 密碼以明文儲存，請勿在公共電腦使用

## 技術

- Chrome Extension Manifest V3
- 原生 HTML / CSS / JavaScript（無框架）
- Shadow DOM 隔離泡泡 UI，避免與頁面樣式衝突
